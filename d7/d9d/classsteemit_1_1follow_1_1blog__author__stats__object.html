Detailed Description

This index is maintained to get an idea of which authors are resteemed by a particular blogger and how frequnetly. It is designed to give an overview of the type of people a blogger sponsors as well as to enable generation of filter set for a blog list.

Give me the top authors promoted by this blog Give me all blog posts by [authors] that were resteemed by this blog
