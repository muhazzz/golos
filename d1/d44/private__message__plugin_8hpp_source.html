39 //
40 // Plugins should #define their SPACE_ID's so plugins with
41 // conflicting SPACE_ID assignments can be compiled into the
42 // same binary (by simply re-assigning some of the conflicting #defined
43 // SPACE_ID's in a build script).

45 // Assignment of SPACE_ID's cannot be done at run-time because
46 // various template automagic depends on them being known at compile
47 // time.
